Ce projet a été développé en utilisant Flask et React
pour le lancer : 
cd backend & python api.py
cd frontend & yarn start
=============================
Fonctionnalités implémentées:
1- Recupération de candidature par son id
2- Récupération de toutes les candidatures
3- Calcul du nombre de candidatures par groupe  formé selon Type_c
4- Mise a jour d'une candidature
5- Recherche d'une candidature par un seule critère (ça peut etre le prenom, nom,...)
6- Création d'une nouvelle candidature
=================================
Nous avons implémenté une simple interface graphique qui comporte un tableau qui affiche toutes les candidatures
en plus d'un formulaire qui permet l'insertion d'une nouvelle candidature
============ 
Vous trouverez les dependances utilisées avec leur version dans le fichier requirements.txt
